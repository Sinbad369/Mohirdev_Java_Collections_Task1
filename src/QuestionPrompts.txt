### **English Prompts:**

#### **ArrayList Tasks 1:**
1. **a)** Using ArrayList, add seven colors and print them.
2. **b)** Change the first element of the colorList into another value.
3. **c)** Write a program that prints the elements of an ArrayList and their size.

#### **ArrayList Tasks 2:**
1. **a)** Create an ArrayList filled with any data of your choice. Then, search for an element in the collection based on a specific value. Display a message on the console indicating whether the searched element was found or not.
2. **b)** Make a copy of the ArrayList.
3. **c)** Write a program to shuffle the elements of the ArrayList.

#### **ArrayList Task 3:**
1. **a)** Write a method to arrange the elements of an ArrayList in reverse order.
2. **b)** Write a method to extract a portion of the ArrayList based on given parameters (int firstIndex, int lastIndex).

#### **ArrayList Task 4:**
1. **a)** Write a method that compares two ArrayList collections element by element and writes the result to another collection.
   Example: ["Green", "Red"] & ["Green", "Black"] -> result: ["Yes", "No"]

#### **ArrayList Task 5:**
1. **a)** Write a method to swap two elements in an ArrayList.
2. **b)** Write a method to merge two ArrayList collections.
3. **c)** Create an ArrayList with a capacity to hold only 3 elements, and after adding 3 elements, increase the size of the collection to hold 6 elements.
